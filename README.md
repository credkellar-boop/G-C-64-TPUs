# G-C-64-TPUs
The G-C-64-TPUs architecture uses a dual-layer network topology for AI workloads. The Private Layer links 16 building block trays, uniting 64 TPUs via 120 dedicated links for isolated, ultra-low latency compute. The Public Layer uses a spine-leaf structure to manage cloud orchestration, GKE interfacing, data ingress from GCS, and secure VPC access.
