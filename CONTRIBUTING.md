# Contributing to Google Suite RS

We welcome pull requests! To ensure a smooth process, please follow these steps:

1. **Fork the repository** and create your branch from `main`.
2. **Run tests** to ensure nothing is broken:
   `cargo test`
3. **Format your code** using the project's config:
   `cargo fmt`
4. **Lint your code** to catch common mistakes:
   `cargo clippy -- -D warnings`

Ensure your commit messages are clear and describe the exact changes made.
